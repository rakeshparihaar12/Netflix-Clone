# RakeshFlix 🎬

A responsive Netflix clone built using HTML and CSS.

## 🚀 Live Demo
https://rakeshflix.netlify.app

## ✨ Features
- Responsive Design
- Netflix Hero Section
- TV Video Section
- Footer Grid Layout

## 🛠️ Technologies Used
- HTML
- CSS
